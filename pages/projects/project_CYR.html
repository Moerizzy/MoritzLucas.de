Clear Your Ride

Development of an android app that can record grievances on cycle paths

by Nele Wichert & Moritz Lucas

In this project, an app was developed that makes it possible to report grievances in the urban cycling infrastructure, to display them, to filter them and to mark them as fixed. The app was developed on the base of the Cordova framework and is therefore basically a website. This was implemented with a Leaflet web map and Ajax queries, which wrote and queried the data on the server into a PostgreSQL database. Unfortunately, communication with the server is no longer possible, as it was taken offline after the University project. The codebase can be downloaded here.
